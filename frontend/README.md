# Stock Tracker Frontend

This folder contains the React frontend for the Stock Tracker application.

## Tech Stack

* React
* TypeScript
* Vite
* React Router

## Folder Structure

```
src/
├── assets/
├── components/
├── hooks/
├── pages/
├── services/
├── App.tsx
└── main.tsx
```

## Available Routes

| Route        | Description        |
| ------------ | ------------------ |
| `/`          | Home page          |
| `/favorites` | Favorite stocks    |
| `/history`   | Stock history      |
| `*`          | 404 Not Found page |

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Status

This is the initial frontend scaffold for the project. Future development will include UI components, API integration, state management, and styling.
